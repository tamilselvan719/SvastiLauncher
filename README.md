# SvastiLauncher

SvastiLauncher is a custom Android launcher built with Jetpack Compose and Room Database.

## Project Motivation

This project was born out of frustration with modern "premium" launchers. Even well-known premium launchers like Nova Launcher have been known to include intrusive elements or be acquired by major ad-tech companies.

I believe that a launcher—the primary interface to your personal device—should be:
1. **Privacy-First**
2. **Ad-Free**
3. **Performant**
4. **Customizable**

## Architecture
- **Jetpack Compose:** For a modern, declarative UI.
- **Hilt (Dependency Injection):** For clean, testable architecture.
- **Room:** For local data persistence.
- **MVVM:** Clear separation of concerns between UI and business logic.
